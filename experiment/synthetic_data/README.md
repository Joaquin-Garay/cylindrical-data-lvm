# Synthetic Data Experiment Series

This folder contains synthetic experiments for comparing latent-variable models on mixed Euclidean and directional data. The experiments are organized as controlled ablations: each dataset changes one or two factors in the data-generating process, then the same candidate models are fit and selected with the same calibration protocol.

The main question is how model selection behaves when the data are generated by either:

- a single-level mixture of Cylindrical distributions, where Euclidean and directional variables can be statistically coupled; or
- a two-layer hierarchical mixture, where a first latent layer selects a Euclidean block modeled as Gaussian and a second latent layer selects a directional subcomponent within each first layer component modeled as von-Mises Fisher distribution.

## Candidate Models

Each synthetic dataset is evaluated with the same model families:

- **Cylindrical mixture**: a mixture of full cylindrical components. This model can represent dependence between the Euclidean block and the directional block through the cross-covariance term $\Sigma_{12}$.
- **Independent Cylindrical mixture**: an instance of the Cylindrical mixture with the Euclidean and directional blocks modeled independently inside each component.
- **Two-layer MoM**: a hierarchical mixture-of-mixtures model with a Gaussian mixture in the first layer and conditional directional mixtures in the second layer.
- **Isolated two-layer MoM**: a two-stage MoM modification that fits the first-layer mixture first, then fits each second-layer mixture using first-layer posterior weights.

For each model family, selection is performed over a grid of component counts using cross-validation and BIC or negative log-likelihood. Runs are repeated from multiple initializations to reduce sensitivity to local optima.

## Experiment 01: Cylindrical Mixture Data-Generating Process

The first experiment uses a mixture of cylindrical distributions as the data-generating process. This is the correctly specified setting for the full cylindrical mixture and a misspecified setting for the hierarchical models.

The ablations vary:

- **Dimension**: low-dimensional 2D experiments for visual diagnostics and 3D experiments matching the main modeling setting.
- **Cluster separability**: low-separability cases test overlapping clusters; high-separability cases test easier recovery.
- **Euclidean-directional dependence**: $\Sigma_{12}=0$ gives an independent baseline, while weak and strong dependence test whether models benefit from explicitly modeling cross-block dependence.
- **Noise**: pure samples are compared against perturbed samples with Gaussian noise in the Euclidean block and renormalized noise in the directional block.

### Interpretation Goals

- Does the full cylindrical mixture recover the correct number of components when it is correctly specified?
- How much performance is lost when the independent cylindrical ablation ignores $\Sigma_{12}$?
- Can a two-layer model approximate single-level cylindrical dependence, or does it over-split the data?
- How sensitive are the selected component counts and scores to noise, separability, and dependence strength?

## Experiment 02: Two-Layer Hierarchical Data-Generating Process

The second experiment uses a two-layer hierarchical mixture as the data-generating process. This is the correctly specified setting for the two-layer MoM and a misspecified setting for the single-level cylindrical mixtures.

The first layer samples a Euclidean Gaussian cluster. Conditional on that first-layer cluster, the second layer samples a directional component. This creates data where the Euclidean structure and directional structure are related through a hierarchy rather than through a single cylindrical component.

The ablations vary:

- **Number of first-layer clusters**: controls the coarse Euclidean grouping.
- **Number of second-layer components per first-layer cluster**: controls within-group directional heterogeneity.
- **Layer separability**: controls whether hierarchy is visually and statistically obvious.
- **Noise**: compares pure hierarchical samples against perturbed samples.

### Planned Variants

| Variant | Description | Purpose |
| --- | --- | --- |
| `02C` | First-layer clusters $K=4$, Euclidean dimension $d=3$, second-layer counts $K_j=\{3,3,3,3\}$, directional dimension $m=3$ | Main balanced hierarchical setting. |

### Interpretation Goals

- Does the two-layer MoM recover the intended first-layer and second-layer component counts?
- Does the isolated two-layer ablation lose information by fitting layers too separately?
- Do single-level cylindrical mixtures approximate hierarchical data by increasing the number of components?
- Which model family gives the clearest trade-off between fit quality, selected complexity, and interpretability?

## Calibration Protocol

For every dataset variant:

1. Generate a pure synthetic dataset from the specified data-generating process.
2. Generate a noisy version using the same latent structure plus controlled perturbations.
3. Fit each candidate model family over its component-count grid.
4. Select configurations using cross-validated BIC or negative log-likelihood.
5. Record all configuration-level scores, the selected model, and the refit score for comparison.

The extensive experiment scripts export configuration-level results to `results/synthetic_data/`, allowing comparisons across model family, data-generating process, noise setting, and ablation axis.
